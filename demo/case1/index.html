Case 1 of routing

The definition
Let's l to be the window.location.hash
Let's o to be the object-content
Let's r to be the o's attribute hash
By default o is hidden
If match proccess matches r with l then the content of o will be displayed

The definition of be displayed means that
  1. if src attribute has a value then let's the content to be c - the result of fetching and processing from the given url
  2. if src attribute is not present then let's the content to be c - the o's children
  3. unhide o
